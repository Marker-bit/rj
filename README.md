# Reading Journal

Reading Journal is an app made in Next.js for reading fans.

## Stack

1. Next.js
2. Typescript
3. Prisma
4. Shadcn/ui
5. Tailwind CSS

## Installation

1. `npm install` - install npm dependecies
2. Fill `.env` file
3. `npx prisma db push` - push schema to database
4. `npm run dev` - start server
